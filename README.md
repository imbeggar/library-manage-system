# javaweb期末大作业--图书管理系统
## 实现功能
1. Cookie-Session用户状态检测
2. 通过邮件找回密码
3. 图书CRUD
4. 用户CRUD
5. 用户组过滤
6. 在线人数，总访问人数统计
7. 使用Graphic绘图类生成验证码

## 开发环境
1. IntellJ IDEA
2. Tomcat8
3. java8
4. mysql5.5
